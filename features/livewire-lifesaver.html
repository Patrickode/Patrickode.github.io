Livewire Lifesaver

Role: Sole Developer
Platform: Unity, C#

Livewire Lifesaver is an enhanced version of my first attempt at the Ludum Dare Compo, for Ludum Dare 46. I was experimenting with a velocity-based movement system on a spherical player before the compo, and I liked it so much I decided to do something with it in the compo.

I revisited Livewire Lifesaver the following summer to polish it a little, but I was so satisfied with my improvements on the movement that I made it a larger project.

  • I noticed that moving against a wall while falling slowed the player's descent. I thought this would be fun, so I made it a full mechanic; I used vector math to adjust the player's velocity when moving against a wall to allow wall-running, wall-climbing, and wall-jumping.
    • I added timers that deaden input when doing each of these, giving the player some leeway and ensuring a smooth, human-friendly experience
  • The current automatically follows wire pieces, without needing a direct, developer-assigned reference to them; this saves a ton of tedium and makes level creation as simple as drag and drop.
  • I wanted to learn Unity's new input system, so I reverse-engineered and customized the code from Unity's interactive rebinding demo; this allowed for controller support and custom control schemes that are preserved between play sessions

Divinity Story Hub

Role: Sole Developer
Platform: HTML, CSS, Jekyll, JavaScript

Divinity is a fantasy story series about gods, angels, demons, and the war between them. I write it in my spare time out of passion. Divinity was initially conceived some time in 2012, and it's grown just as much as I have since. I built this website from the ground up so I could put Divinity's chapters online, to share them with others more easily.

  • I got fed up with the lack of customization I had on WordPress and other website builders, so using my designs from those sites as my specification, I built this website starting from an empty html file.
  • I didn't want to copy and paste my work into the website every time I updated a chapter, so I embedded the source google doc into the site. While trying to figure out how to style that embedded doc with CSS, I discovered a way to extract that doc's contents and insert it into the website by itself with a CORS request; this allows for live updates and full freedom with CSS.
  • I added Bookmark and Light/Dark mode buttons, programmed with JavaScript, to make reading as convenient as I could for visitors.
